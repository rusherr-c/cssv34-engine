# CSSv34 Engine
Modified version of Source Engine 2013 with CSS v34 server-connection support<br>

Partially used code from:
* https://github.com/uvbs/source-2007
* https://github.com/ValveSoftware/source-sdk-2013
* https://github.com/mapbase-source/source-sdk-2013
* https://github.com/nillerusr/source-engine

Compiling can be done with VS 2022, 2026
## Requirements:

MFC C++ Library (can be downloaded in Visual Studio Installer)<br>
Strawberry Perl https://strawberryperl.com (if you want to compile shaders)
